90 /* A weights connect op to perform lateral processing of different

92  neighbors according to a configurable linear function of distance
93  (see parameters below). This op is aware of node geometry: if node
94  rows have been specified (see mint_node_rows), neighbours are
95  calculated in two dimensions. The configurable parameters allow
96  several kinds of lateral processing, such as pure lateral
97  inhibition and center-surround effects (Mexican hat or reversed
98  Mexican hat, see examples).
99 
100  State variables: none required.
101 
102  Parameters: 0: Connection value for nearest neighbors (positive or
103  negative).
104 
105  1: Maximum distance. No input will be sent to nodes
106  further than this.
107 
108  2: Connection value at maximum distance (positive or
109  negative).
110

113 
114  EXAMPLES: Lateral inhibition of -1 sent to nearest neighbors only:
