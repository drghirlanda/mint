Run a simple genetic algorithm starting with a given network.

At each iteration, one mutated network is created, the performance of both the original and the mutated network is assessed, and the better network is retained for next iteration. The best network will be available as net1 after the function returns.

Parameters
net1The initial network
perfFunction that performs network evaluation and returns the result as a float. The function attempts to maximize this function.
mutFunction that mutates a network.
targetThe algorithm will stop if this value of performance is reached
maxThe algorithm will stop after max iterations
Returns
The performance value attained.
